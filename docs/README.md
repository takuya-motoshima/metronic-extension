# demo
This demo uses metronic_v8.1.2.

## Getting Started
1. Install front-end dependencies.
    ```sh
    npm install
    ```
1. Build the front-end module.
    ```sh
    npm run build
    ```
1. Install server-side dependencies.
    ```sh
    cd server
    npm install
    ```
1. Start the server.
    ```sh
    npm start
    ```
1. Open docs/index.html in a browser.
